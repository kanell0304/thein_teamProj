# 모먹지 (Momeogji)

> 카카오톡 채팅방 안에서 대화만으로 모임 맛집을 추천받고, 위치를 확인하고, 예약까지 한 번에 끝내는 서비스

> **주의**: 본 문서는 초안(draft)이며, 기획 및 개발 진행 상황에 따라 지속적으로 수정됩니다.

## 📌 프로젝트 소개

`모먹지`는 채팅방에서 오가는 대화 속 키워드를 자동으로 추출하고, 이를 바탕으로 AI가 근처 음식점을 추천해주는 서비스입니다. 추천된 음식점의 위치는 지도로 바로 확인할 수 있고, 예약이 가능한 경우 예약까지 대신 진행합니다.

카카오톡 단체 채팅방에서 점심·저녁 약속을 잡을 때 반복되는 번거로운 의사결정 과정(어디서 만날지, 무엇을 먹을지, 어디를 예약할지)을 채팅창 안에서 압축해서 해결하는 것이 목표입니다.

### 제안 배경

- 직장 동료·친구 모임에서 식사 장소를 정하는 과정은 시간대 조율보다 훨씬 복잡하고 소모적입니다.
- 매일 정해야 하는 점심 메뉴, 동아리/프로젝트 모임 식사 자리는 특히 의사결정 피로도가 큽니다.
- 채팅창을 벗어나 지도 앱을 켜고 직접 검색하는 과정을 없애고, 대화 흐름 안에서 추천 → 확인 → 예약까지 이어지도록 합니다.

### 기대 효과

1. **채팅 안에서 완결되는 경험** — 지도 앱으로 이탈하지 않고 채팅방 안에서 메뉴 선정 전 과정을 진행
2. **소외되기 쉬운 의견도 반영** — 대화 속 키워드 기반 추천으로, 적극적으로 의견을 내지 않는 참여자의 취향도 자연스럽게 반영
3. **모임 주최자의 부담 감소** — 카드 형태로 제공되는 선택지를 고르기만 하면 되고, 선택 결과가 참여자 전원에게 자동 공유

## 🍽️ 핵심 기능

1. **키워드 추출** — 채팅방 대화 내용에서 음식/지역/분위기 등과 관련된 키워드를 추출
2. **음식점 추천** — 추출된 키워드를 OpenAI API에 전달하여 근처 추천 음식점 목록을 생성
3. **위치 표시** — 추천된 음식점의 주소를 카카오맵 API로 조회하여 채팅방 안에 위치를 표시
4. **자동 예약** — 예약이 가능한 음식점인 경우, 서비스가 대신 예약을 진행

## ⚠️ 리스크

- **개인정보 부담**: 추천 정확도를 높이기 위해 활용 가능성이 있는 개인 데이터(마이데이터 등)에 대해 사용자 사전 동의가 필요하며, AI가 이를 활용하는 것에 부담을 느낄 수 있음
- **AI 추천 정확도**: 부정확한 추천이 반복되면 기능 자체에 대한 신뢰뿐 아니라 AI/서비스 전반에 대한 신뢰 저하로 이어질 수 있음
